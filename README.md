# 🌟 SkyBlaster: Hang Glider Asteroids

**Fly. Shoot. Survive.**  
A fun, lightweight browser-based shoot-'em-up where you pilot a hang glider through a stunning golden-hour desert sky while blasting incoming asteroids!

**[Play Now → https://tommy0x2a.github.io](https://tommy0x2a.github.io)**

![Game Screenshot](https://user-images.githubusercontent.com/XXXXXX/skyblaster-screenshot.png)  
*(Replace with a real screenshot or GIF once you capture one)*

## 🎮 How to Play

- **Mouse** → Move up/down to steer the hang glider  
- **Spacebar** → Fire glowing energy bolts ⚡  
- Destroy as many asteroids as you can while the mountains scroll beneath you!

The glider experiences realistic **turbulence**, so stay alert!

## ✨ Features

- Smooth mouse-controlled flight with natural banking and turbulence
- Beautiful glowing, pulsing sun with soft halo (pure CSS)
- Fast-moving energy bolts with glowing effects
- Randomly spawning asteroids flying in from the right
- Real-time HUD showing:
  - Distance Flown
  - Altitude
  - Score (10 points per asteroid destroyed)
- Responsive full-screen desert mountain background that scrolls endlessly
- Zero dependencies – just one `index.html` file

## 🛠 Tech Stack

- Pure HTML5 + CSS3 + Vanilla JavaScript
- `requestAnimationFrame` game loop
- CSS animations & gradients for the sun and bolts
- No external libraries or frameworks
